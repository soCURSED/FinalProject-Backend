**User:**
- Register
- Login
- Get all users
- Get user by Id
- Update user
- Delete user
- Add cart to user
- Add product to user's cart

**Cart:**
- Get all carts
- Get cart by Id
- Update cart payment method
- Delete cart

**Product:**
- Get all products
- Get product by product Id
- Update product
- Delete product
